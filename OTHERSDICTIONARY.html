ALL Technical Dictionary

Antistatic Wrist Strap - A safety device used when repairing or assembling computers. It prevents electrostatic discharge (ESD) from damaging components.
BIOS - Basic Input/Output System. It is firmware that initializes and tests hardware during the boot process before the operating system loads.
CPU - Central Processing Unit. Known as the brain of the computer, it executes instructions and processes data.
CSS - Cascading Style Sheets. A stylesheet language used to control the design, layout, and presentation of web pages.
Firewall - A network security system that monitors and controls incoming and outgoing traffic to protect systems from threats.
Hardware - The physical components of a computer system such as the CPU, RAM, motherboard, and storage devices.
HDD - Hard Disk Drive. A traditional storage device that uses spinning disks to read/write data.
HTML - HyperText Markup Language. The standard markup language used for creating and structuring web pages.
IP Address - Internet Protocol Address. A unique identifier assigned to each device connected to a network for communication purposes.
JavaScript - A programming language mainly used for adding interactivity, dynamic content, and functionality to web pages.
LAN Cable - Local Area Network cable. Used to connect computers and devices within a network for wired communication.
Motherboard - The main circuit board of a computer that connects and allows communication between all hardware components.
Multimeter - An electronic measuring instrument used to test voltage, current, and resistance in electrical circuits.
Ping - A network utility used to test connectivity between devices by sending packets of data and measuring response time.
PSU - Power Supply Unit. Provides power to all components of a computer by converting electricity from the outlet into usable power for hardware.
RAM - Random Access Memory. Temporary memory used by the computer to store data and instructions that are currently being used or processed.
Screwdriver - A tool used for tightening or loosening screws when assembling or repairing computers and electronic devices.
Software - Programs, applications, and operating systems that run on a computer and allow users to perform tasks.
SSD - Solid State Drive. A storage device that uses flash memory to store data, providing faster speed than traditional HDDs.
CMOS Battery - A small battery on the motherboard that provides power to store BIOS settings.
DVI - Digital Visual Interface. A video display interface used to connect a video source to a display device.
Ethernet - A networking technology for wired local area networks (LANs) that uses cables to connect devices.
Firewall - A network security system that monitors and controls incoming and outgoing network traffic.
HDMI - High-Definition Multimedia Interface. A digital interface used for transmitting audio and video signals.
Heat Sink - A device that dissipates heat away from computer components, usually from the CPU or GPU.
Hub - A basic networking device that connects multiple devices in a LAN but broadcasts data to all devices.
IP Address - A numerical label assigned to each device connected to a computer network.
NIC - Network Interface Card. A hardware component that allows a device to connect to a network.
PATCH Panel - A mounted hardware assembly with ports used to connect and manage cables in a network.
POST - Power-On Self Test. A diagnostic testing sequence run by a computer's BIOS to check hardware components before booting the OS.
RJ11 - A connector typically used for telephone lines and modems.
RJ45 - A standard connector for Ethernet networking cables.
Router - A device that forwards data packets between computer networks, providing internet connectivity.
Switch - A network device that connects devices within a LAN and forwards data only to the intended recipient.
UPS - Uninterruptible Power Supply. A backup power source that keeps computers running during power outages.
VGA - Video Graphics Array. An analog interface used to connect monitors to computers.
WAN - Wide Area Network. A telecommunications network that extends over a large geographic area.
AJAX - Asynchronous JavaScript and XML. Technique to fetch data in the background without reloading a page.
Adobe XD - Adobe Experience Design. Vector-based UI/UX design and prototyping tool by Adobe.
After Effects - Adobe After Effects. Software for motion graphics and visual effects.
Angular - Angular Framework. Google-maintained TypeScript framework for large-scale web apps.
Aspect Ratio - Width-to-Height Ratio. Proportional relationship between width and height of media.
Babel - Babel Transpiler. JavaScript compiler that converts modern JS/TS into browser-compatible code.
Bootstrap - Bootstrap CSS Framework. Popular CSS & JS toolkit for responsive, mobile-first websites.
Branch - Git Branch. Parallel line of development in a Git repository.
CI/CD - Continuous Integration / Continuous Deployment. Practices that automate testing and deployment of code.
CSS3 - CSS3. Latest CSS specification with features like Flexbox, Grid, animations and variables.
Canvas - HTML Canvas. Bitmap drawing area in HTML5 used for 2-D games, graphs, and image editing.
Codec - Coder-Decoder. Software or hardware that compresses/decompresses audio or video.
Commit - Git Commit. Snapshot of changes saved to a Git repository history.
Container - Software Container. Isolated runtime environment that bundles application code with its libraries.
Docker - Docker Container Platform. Tool for packaging applications and dependencies into portable containers.
Express - Express JS. Minimal Node.js web application framework for APIs and servers.
Figma - Figma Design Tool. Browser-based UI/UX design and prototyping platform for collaborative work.
FPS - Frames Per Second. Number of frames displayed each second in video or animation.
Fetch API - Fetch API. Modern JavaScript interface for making HTTP requests and handling promises.
GIF - Graphics Interchange Format. Image format supporting 256 colors and simple animations.
Git - Git Version Control. Distributed version-control system for tracking code changes.
GitHub - GitHub Platform. Cloud service for hosting Git repositories and collaboration.
GitLab - GitLab Platform. DevOps lifecycle tool providing Git hosting and CI/CD pipelines.
HTML5 - HTML5. Latest major version of HTML introducing semantic tags, audio/video elements, and Canvas.
Illustrator - Adobe Illustrator. Vector graphics editor used for logos, icons, and illustrations.
JPEG - Joint Photographic Experts Group. Common lossy image format balancing quality and size.
JSON - JavaScript Object Notation. Lightweight data-interchange format easy for humans and machines to read/write.
Kubernetes - Kubernetes. Open-source system for automating deployment, scaling, and management of containerized apps.
LESS - Leaner Style Sheets. CSS pre-processor with variables and modularization.
MP3 - MPEG-1 Audio Layer III. Popular lossy audio format (music, podcasts).
MP4 - MPEG-4 Part 14. Digital multimedia container format for storing video and audio.
Merge - Git Merge. Combines changes from different branches into one.
Mockup - Design Mock-up. High-fidelity static design showing final look before coding.
NPM - Node Package Manager. Default package manager for Node.js ecosystems.
Node.js - Node JavaScript Runtime. Server-side JavaScript runtime built on Chrome's V8 engine.
OGG - Ogg Vorbis. Open, patent-free container often used for Vorbis audio.
PNG - Portable Network Graphics. Raster image format supporting transparency and lossless compression.
Photoshop - Adobe Photoshop. Industry-standard raster graphics editor for images and digital art.
Premiere Pro - Adobe Premiere Pro. Professional video editing software.
Prototype - Interactive Prototype. Clickable model simulating user flow for testing UX.
REST API - Representational State Transfer API. Architectural style for stateless web services using HTTP methods.
React - React JS. JavaScript library for building user interfaces with components and virtual DOM.
Resolution - Pixel Dimensions. Number of pixels in each dimension that can be displayed.
SASS - Syntactically Awesome Style Sheets. CSS pre-processor adding variables, nesting, mixins and functions.
SVG - Scalable Vector Graphics. XML-based vector image format for sharp graphics at any resolution.
SVG (File) - Scalable Vector Graphics File. Vector image file rendered via XML; ideal for icons and logos.
Tailwind - Tailwind CSS. Utility-first CSS framework for rapidly building custom designs.
TypeScript - TypeScript. Superset of JavaScript adding static types and modern syntax.
Vue - Vue JS. Progressive JavaScript framework focused on the view layer and reactivity.
WAV - Waveform Audio File Format. Uncompressed audio format with high fidelity.
WebGL - Web Graphics Library. JavaScript API for rendering interactive 3-D graphics in the browser using GPU.
Webpack - Webpack Module Bundler. Build tool that bundles JavaScript, CSS, images into optimized assets.
Wireframe - UI Wireframe. Basic schematic layout of a web page or app screen.
Yarn - Yarn Package Manager. Fast, secure JavaScript package manager alternative to NPM.
package.json - Node Package Manifest. File listing project metadata, dependencies, and scripts for Node projects.
